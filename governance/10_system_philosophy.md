# System Philosophy

EdDesk is a **structured CMS**, not a free-form page builder.

---

# Principles

| Principle      | Meaning                        |
| -------------- | ------------------------------ |
| Predictability | behavior must be deterministic |
| Safety         | avoid schema damage            |
| Explicit Rules | no hidden behavior             |

---

# Non Goals

EdDesk is NOT:

* drag-and-drop website builder
* visual layout editor
* dynamic CMS
